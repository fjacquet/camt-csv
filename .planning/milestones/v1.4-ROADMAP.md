## Key Decisions

- Folder mode is non-recursive; file extension filtered per parser
- PDF folder mode consolidates to one CSV (existing behavior promoted to default)
- `--format` flag remains available; only default changes to icompta
- `batch` subcommand and `--batch` flag removed with no deprecation period
- `osExitFn` package variable used in FolderConvert for testable `os.Exit` injection
- EmptyDirectory asserts exit code 2 (consistent with BatchManifest contract)
